# PumaLand - Servidor de Minecraft

Sitio web oficial del servidor PumaLand con página principal y tienda.

## Estructura del Proyecto

```
/
├── index.html           # Página principal
├── Tienda.html         # Página de la tienda
├── .env                # Variables de entorno
├── .gitignore         # Archivos ignorados por Git
└── assets/            # Recursos del proyecto
    ├── css/           # Hojas de estilo
    │   ├── style.css      # Estilos de la página principal
    │   └── tienda.css     # Estilos de la tienda
    ├── js/            # Scripts JavaScript
    │   ├── script.js      # Script de la página principal
    │   └── tienda.js      # Script de la tienda
    └── images/        # Todas las imágenes del proyecto
        ├── Bannerps.png
        ├── Evento1-6.png
        ├── HomeLogodeinicio.png
        ├── LogoPL.png
        ├── Nosotros2.png
        ├── Pregunta.png
        └── SpawnimagenInicio.png
```

## Características

- 100% Responsive (móviles, tablets y escritorio)
- Código limpio y organizado
- Optimizado para SEO
- Animaciones suaves y efectos visuales
- FAQ con acordeón interactivo
- Slider de banners en la tienda
- Efectos de nieve navideña

## Navegación

- **index.html**: Página principal con información del servidor
- **Tienda.html**: Tienda con rangos y productos

## Tecnologías

- HTML5 semántico
- CSS3 con Flexbox y Grid
- JavaScript ES6+
- Diseño responsive con media queries

## Notas

- Las rutas de imágenes usan referencias relativas desde `assets/images/`
- Los estilos CSS están organizados por secciones
- Los scripts JavaScript están modularizados
- El diseño visual se mantiene idéntico al original
